Quick Start

Last updated: January 7, 2020

Background

ORCID

ORCID is an open, independent, non-profit organization which aims to connect global research and researchers. ORCID offers a registry of reseacher identifiers to uniquely indentify contributors to research, scholarship, and innovation activities.

ORCID offers an API that allows one to access the ORCID registry, including reading from and writing to ORCID records.

This project uses a Python Wrapper of ORCID API to access ORCID record.

PubMed

PubMed is a free search engine accessing MEDLINE, life science journals and online books. PubMed is a free resource that is developed and maintained by the National Center for Biotechnology Information (NCBI), at the U.S. National Library of Medicine (NLM), as part of the Entrez system of information retrieval, located at the National Institutes of Health (NIH).

A distinctive feature of MEDLINE is that the records are indexed with NLM Medical Subject Headings (MeSH).

This project uses Entrez package provided by Biopython to access NCBI.

git clone git@github.com:mlion0200/educase_TVB_braintumor.git

Download Data

Data contains functional and structural connectivity matrices of 25 brain tumor patients and 11 healthy control participants before surgery, and 18 brain tumor patients and 10 healthy control participants after neurosurgery.

Data is in demo-data folder in the GitHub repository.

Data can also be found on OpenNeuro. The pre-operative data can be found here, and post-operative data can be found here.

To install TVB code on your machine, we recommend you to first create a dedicated Python environment using Anaconda command.

conda create --name tvb python=2.7

conda activate tvb


Run pip from command line to install tvb released packages:

pip install tvb-library

pip install tvb-framework
